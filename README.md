# SexToyScriptViewer

大人のオモチャ用の連動スクリプトを見やすくチャート(グラフ)表示するWindowsアプリです。

## 機能

- 同時に複数のスクリプトを表示可能
- 音声や動画ファイルを読み込んで再生することでチャート上に再生位置の目印を表示でき、音声を聞きながらスクリプトの確認が可能

## 対応スクリプト形式

- funscript
- U.F.O. SA/A10サイクロンSA用CSV
- U.F.O. TW用CSV
- タイムローター用CSV

A10ピストン用CSVは現在非対応です。(今後対応予定)

## 使用ライブラリ
[OxyPlot 2.0.0](https://github.com/oxyplot/oxyplot)
